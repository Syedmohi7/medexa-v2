## 🌟 Overview

**Medexa** is an AI-powered healthcare virtual assistant designed to make healthcare **smarter, faster, and more accessible**.  

It helps users:
- Analyze symptoms 🧠  
- Understand medical reports 📄  
- Get AI-based health insights 🤖  
- Access doctors and nearby hospitals 🏥  

All in one unified platform.

---

## 🎯 Key Features

- 🧠 **AI Symptom Analysis**  
  Predicts possible diseases based on user symptoms  

- 📊 **Health Dashboard**  
  Tracks BMI, hydration, calories, and activity  

- 📄 **Medical Report Analysis (OCR + AI)**  
  Upload prescriptions/reports and get insights  

- 🤖 **AI Chat Assistant (Jozo AI)**  
  Interactive chatbot for health queries  

- 👨‍⚕️ **Doctor Consultation**  
  Integration with platforms like Apollo & MediBuddy  

- 🏋️ **Workout & Wellness Plans**  
  Personalized fitness recommendations  

- 📍 **Nearby Hospitals**  
  Location-based emergency support  

- 🔐 **Authentication & Admin Panel**  
  Secure login + admin analytics dashboard

## 🛠️ Tech Stack

### 💻 Frontend
- Next.js  
- Tailwind CSS  

### ⚙️ Backend
- Node.js  
- Express.js  

### 🗄️ Database
- MongoDB  

### 🤖 AI / ML
- Python (Disease Prediction Model)  
- OCR for report analysis  

### ☁️ Deployment
- Vercel (Frontend)  
- Render (Backend)  
- Cloudinary (Media Storage)  

---

## 🧠 Architecture

- Frontend (Next.js) → UI/UX & API calls  
- Backend (Express) → Business logic & APIs  
- ML Model (Python) → Prediction engine  
- MongoDB → Data storage
